Explorer Overview

Use the search above, or choose one of the lookup tools below.

Archive
Suggested use: enter a block height to load its block overview, use “Block Orders” to inspect all orders in that block, or enter a Noso address to check its V1 balance. Numeric searches from 0 through 180031 are treated as block heights. Numeric values from 00 through 99 can also be looked up in the Governance section.
Block lookupCalls getblocksinfo and shows timestamps, hash, target, solution, minted address, reward, and fees.
Block ordersCalls getblockorders and renders each order as a readable card with sender, receiver, amount, fee, and reference.
Address lookupCalls getaddressbalance and preserves the current balance / incoming / outgoing logic from your existing page.
Pseudo walletPaste multiple Noso addresses to total them together as a read-only wallet view. Each address is queried individually with getaddressbalance.
Governance lookupSearches the fixed list of 100 current GVT governance slots and can also load a selected address balance.
Exchange addressesTags known exchange deposit/collection addresses in address lookup, block orders, and pseudo wallet results: XeggeX, NonKYC, and Azbit.

Pseudo Wallet Calculator

Paste 2 or more NosoV1 addresses and the page will total the balances into one read-only wallet-style view.

Enter a block height from 0 through 180031.
Enter a block height to list all orders in that block. Example: final block 180031.
Enter a Noso address to check its V1 balance.
Paste one address per line, or separate addresses with commas, spaces, or semicolons. The page will query each address and total the current balances together without storing anything on a server.
Paste a list of addresses to calculate a combined V1 balance.
Select a governance slot below, or search by slot/address.
Known addresses can be looked up directly or loaded into the Pseudo Wallet Calculator. Other historical exchanges are listed for reference where the NosoV1 address is unknown.
